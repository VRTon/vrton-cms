# VRTon Website - Asset Directory

This directory contains all the assets for the VRTon website.

## Required Assets

### Logo
- `logo.png` - Main VRTon logo (recommended size: 180x180px or higher resolution)

### VRChat Badge
- `vrchat-placeholder.png` - Square image for VRChat experience badge (120x120px)

### Event Images (3 images per year)
**2023 Event:**

## Image Specifications

- **Logo**: PNG format with transparent background preferred
- **VRChat Badge**: Square aspect ratio (1:1)
- **Event Images**: Landscape orientation recommended (4:3 or 16:9 aspect ratio)
- **Format**: JPG or PNG
- **Quality**: High resolution for retina displays

## Notes

The website includes automatic placeholder generation if images are missing, but please replace them with actual images for production use.
